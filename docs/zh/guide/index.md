# 指南

## 简介
OhMySkills 是您获取高质量 AI 代理技能和提示词的首选之地。无论您使用的是 **Claude**、**ChatGPT**、**Codex** 还是 **Antigravity**，我们都有资源来增强您的工作流程。

## 如何使用

### 浏览技能
访问 [技能广场](/zh/skills/) 浏览我们的收藏。您可以按类别筛选或搜索特定关键字。

### 下载技能
每个技能卡片都有一个下载按钮。点击它将下载一个 ZIP 文件，其中包含：
- 提示词文件（Markdown 或 JSON）
- 元数据文件
- 针对您所选平台的安装指南 (`INSTALL.md`)。

### 配合 Antigravity 使用
OhMySkills 对 **Antigravity** 提供一流支持。
1. 在平台选择器中选择 "Antigravity"（如果可用）。
2. 下载技能。
3. 将技能包直接导入您的 Antigravity 工作区。
