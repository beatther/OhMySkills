# Guide

## Introduction
OhMySkills is your premier destination for high-quality, curated skills and prompts for AI agents. Whether you are using **Claude**, **ChatGPT**, **Codex**, or **Antigravity**, we have resources to supercharge your workflow.

## How to Use

### Browsing Skills
Visit the [Skills Plaza](/skills/) to browse our collection. You can filter by category or search for specific keywords.

### Downloading Skills
Each skill card has a download button. Clicking it will download a ZIP file containing:
- The prompt file (Markdown or JSON)
- A metadata file
- An installation guide (`INSTALL.md`) tailored to your selected platform.

### Using with Antigravity
OhMySkills has first-class support for **Antigravity**.
1. Select "Antigravity" in the platform selector (if available).
2. Download the skill.
3. Import the skill package directly into your Antigravity workspace.
