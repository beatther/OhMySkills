# Categories

The following categories are available in our library:

- **Research**: Agents dedicated to deep diving into topics.
- **Coding**: Assistants for refactoring, generating, and debugging code.
- **Writing**: Creative partners for storytelling and copy.
- **Data Science**: Tools for analysis and visualization.
- **Design**: Critics and generators for UI/UX.
- **Productivity**: General helpers for onboarding and organization.

Visit the [Skills Plaza](/skills/) to filter by these categories.
